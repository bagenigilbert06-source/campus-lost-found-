ROUTE MAP
================================================================================

PUBLIC ROUTES (No auth required)
  /                  → Home (with PublicNavbar)
  /aboutUs          → About page
  /contact          → Contact page
  /allItems         → Browse items

AUTH ROUTES (Logged-in users redirected)
  /signin           → Sign in (AuthGuard prevents access if logged in)
  /register         → Sign up (AuthGuard prevents access if logged in)
  /admin-login      → Admin login

DASHBOARD ROUTES (Authenticated users only)
  /app/dashboard    → Dashboard home (7-day overview)
  /app/search       → Advanced item search with filters
  /app/post-item    → Post a found item
  /app/post-lost-item → Post a lost item
  /app/my-items     → Your postings
  /app/messages     → Inbox with 3 tabs (All/Unread/Read)
  /app/activity     → Activity timeline with 5 filters
  /app/profile      → Account settings

ADMIN ROUTES (Admins only)
  /admin            → Admin dashboard
  /admin/inventory  → Item management
  /admin/claims     → Claims review
  /admin/reports    → Reports & analytics

LEGACY ROUTES (Still work, backward compatible)
  /dashboard        → Old dashboard (uses UserLayout)
  /profile          → Old profile (uses UserLayout)
  /myItems          → Old items list (uses UserLayout)

================================================================================
NAVIGATION FLOW
================================================================================

UNAUTHENTICATED USER JOURNEY
  ↓
  Visit "/" → Sees PublicNavbar with "Sign In" / "Sign Up"
  ↓
  Click "Sign Up" → Redirected to "/register"
  ↓
  Create account → Automatically redirected to "/app/dashboard"
  ↓
  Now sees DashboardNavbar and DashboardSidebar
  ↓
  Can access all /app/* features

AUTHENTICATED USER JOURNEY
  ↓
  Dashboard Home → View stats, recent activity, quick links
  ↓
  Search Items → Find items with filters (type, category, status)
  ↓
  Post Item → Share a found/lost item
  ↓
  My Items → Manage your postings
  ↓
  Messages → Check inbox, reply to messages
  ↓
  Activity → See timeline of all your actions
  ↓
  Profile → Update account info and preferences

ADMIN JOURNEY
  ↓
  Login with admin email → Redirected to "/admin"
  ↓
  Admin Dashboard → Manage users, items, claims
  ↓
  Cannot access /app/* (redirected to /admin)
