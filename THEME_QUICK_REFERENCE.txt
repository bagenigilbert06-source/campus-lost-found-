================================================================================
COMMON ISSUES & SOLUTIONS
================================================================================

If you see dark backgrounds:
  - Clear browser cache (Ctrl+Shift+Delete or Cmd+Shift+Delete)
  - Hard refresh the page (Ctrl+Shift+R or Cmd+Shift+R)
  - Restart the development server

If text is hard to read:
  - Check that you're using the light theme files
  - Verify no dark: classes are present in components
  - Ensure index.css is properly loaded

If responsive layout is broken:
  - Check breakpoints in Tailwind config
  - Verify flex and grid classes are present
  - Test on actual mobile device

If colors look off:
  - Verify hex color values match (#10b981, #059669, etc.)
  - Check that CSS classes are properly applied
  - Clear CSS cache and rebuild
